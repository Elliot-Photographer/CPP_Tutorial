### Basic data types 
|  Data types  | size |          Description          |
|:------------:|:----:|:-----------------------------:|
|    boolean   |1 byte|  Stores true or false Values  |
|     char     |1 byte|Stores single Character/letter/number, or ASCII values|
|     float    |4 byte|Stores fractional numbers,containing one or more decimals. Sufficient for storing 6-7 decimal digits|
|      int     |2-4 byte| stores whole numbers , without decimal|
|    double    |8 byte|like float but sufficient for storing 15 decimal digits|


if you Don't want to specify the type you can write auto in the place of the type so the compiler chooses what is suitable.
